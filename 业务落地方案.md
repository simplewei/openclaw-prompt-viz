## 二、仍需配置的内容（关键步骤）

2. **钉钉MCP Server配置（已完成）**
   你已提供 Streamable HTTP URL 并已配置到 `mcporter`。

#### 钉钉AI表格（多维表）配置：
1. 访问：https://mcp.dingtalk.com/#/detail?mcpId=9555&detailType=marketMcpDetail
2. 点击 **「获取 MCP Server 配置」**
3. 复制 **Streamable HTTP URL**
4. 执行：
   ```bash
   mcporter config add dingtalk-ai-table --url "YOUR_STREAMABLE_HTTP_URL"
   ```
   或设置环境变量：
   ```bash
   export DINGTALK_MCP_URL="YOUR_STREAMABLE_HTTP_URL"
   ```

#### 钉钉文档配置（可选，用于生成销售文档）：
1. 访问 DingTalk Docs MCP 广场获取 URL
2. 配置：
   ```bash
   mcporter config add dingtalk-docs --url "YOUR_DOCS_URL"
   ```
   or
   ```bash
   export DINGTALK_MCP_DOCS_URL="YOUR_DOCS_URL"
   ```

> 📌 **注意**：`get_document_content_by_url` 目前处于灰度阶段，可能不可用。但创建/搜索文档应正常。

### 2. 钉钉机器人Webhook（可选）

如需自动发送到期提醒到钉钉群：
1. 在钉钉群添加 **自定义机器人**
2. 复制 Webhook URL
3. 设置环境变量：
   ```bash
   export DINGTALK_WEBHOOK="https://oapi.dingtalk.com/robot/send?access_token=XXX"
   ```
   或在脚本中直接修改 `CONFIG.DINGTALK_WEBHOOK`

- 季度版 ¥79.9（90天）← 推荐

### 🔄 自动化流程

#### 1. 客户下单后（闲鱼）
- 手动或自动在 **订单表** 创建记录
- 同时在 **客户表** 创建客户（如果不存在）
- 系统自动计算到期日期并发送邀请到钉钉群

#### 2. 日常管理
- 每日运行 `business-automation.js`
  - 更新 **收入统计**
  - 检查3天内到期的订单
  - 生成提醒（可选钉钉通知）

#### 3. 客户续费
- 在订单表新建记录，套餐ID关联套餐表
- 到期日期自动顺延
- 客户群权限延期（手动或未来自动化）
