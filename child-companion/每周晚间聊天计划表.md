# 晚间聊天计划表 - 每周模板

## 📋 使用说明

**目标**：每天10-15分钟高质量对话，记录成长点滴
**时间**：建议晚饭后7:00-8:00或睡前15分钟
**方式**：家长与孩子一对一，轻松氛围，避免说教
**记录**：聊完后立即记录关键点到child-companion系统

## 🗓️ 每日对话模板

### 周一：情绪工作坊 😊

**开场**："我们来玩情绪猜猜看！"

**核心问题**（选2-3个）：
1. "今天你的心情像什么天气？晴天/多云/下雨/打雷？"
2. "今天什么时候你最开心？什么时候有点难过/生气？"
3. "如果开心可以打分1-10分，你打几分？为什么？"
4. "今天有没有什么让你紧张的事？怎么克服的？"
5. "如果你是小动物，今天想当什么？为什么？"

**互动游戏**：
- "情绪卡片"：用卡片让孩子选代表今天心情的颜色/表情
- "情绪故事"：编一个关于开心小精灵/悲伤小怪物的故事

**记录要点**：
```
## 周一情绪工作坊 [日期]
- 今日心情：____分（1-10）
- 情绪词汇：____、____、____
- 情绪触发点：__
- 应对方式：__
- 家长回应：__
- 观察：__
```

---

### 周二：学习探索日 📚

**开场**："今天学到了什么新本领？"

**核心问题**（选2-3个）：
1. "今天哪节课最好玩？为什么？"
2. "今天你有提问吗？问了什么问题？"
3. "今天哪个字/数字/单词最难？现在会了吗？"
4. "如果变成小老师，你想教什么？"
5. "今天有没有让老师表扬？什么好事？"

**互动环节**：
- "教家长一课"：让孩子教你今天学到的1个知识点
- "超级问题"：每人提1个今天想到的有趣问题

**记录要点**：
```
## 周二学习探索 [日期]
- 最有趣的课：____
- 提问次数：____次（0/1/≥2）
- 今日新学：____
- 困难点：____
- 成就感事件：__
- 兴趣线索：__
```

---

### 周三：社交时光 👫

**开场**："今天和朋友们玩了什么好游戏？"

**核心问题**（选2-3个）：
1. "今天和谁一起玩了？玩了什么？"
2. "今天有没有帮同学/被同学帮？"
3. "有没有看到同学难过/生气？你怎么做的？"
4. "如果朋友哭了，你会怎么办？"
5. "你最喜欢和哪个朋友玩？为什么？"

**情景讨论**（假设场景）：
- "如果好朋友抢了你的玩具，你会？"
- "如果看到同学被欺负，你会？"
- "怎么让不开心的小朋友开心起来？"

**记录要点**：
```
## 周三社交时光 [日期]
- 主要玩伴：____
- 合作/冲突事件：__
- 同理心表现：__
- 领导力表现：__
- 社交挑战：__
- 家长建议：__
```

### 周四：创意无限 🎨

**开场**："我们来编个疯狂故事吧！"

**核心问题**（选2-3个）：
1. "如果用3个词形容今天，是哪3个？"
2. "今天有没有什么奇怪/有趣的想法冒出来？"
3. "如果你能发明一个东西，今天想发明什么？"
4. "如果你是动物园园长，你会放什么动物？为什么？"
5. "今天看到了什么颜色/形状让你觉得美？"

**创意游戏**：
- "故事接龙"：每人说1-2句，编一个完整故事
- "世界末日/世界完美"：想象如果世界变成__样
- "新星球冒险"：描述一个新发现的星球

**记录要点**：
```
## 周四创意无限 [日期]
- 3词形容：____、____、____
- 疯狂想法：__
- 想象力评分：1-10分
- 创造性事件：__
- 潜在兴趣：__
- 家长鼓励方式：__
```

### 周五：家庭连接 👨‍👩‍👧‍👦

**开场**："今天有没有想爸爸妈妈？"

**核心问题**（选2-3个）：
1. "今天什么时候最想家/想爸妈？"
2. "你最喜欢我们家的什么？（某个习惯/物件/角落）"
3. "如果家庭是个足球队，你觉得自己踢什么位置？"
4. "今天有没有什么秘密想告诉我们？"
5. "我们什么时候一起做件特别好玩的事？"

**感恩环节**：
- "今天最感谢的3件事/3个人"
- "今天我们一起做的最好的1件事是什么？"

**未来计划**：
- "周六/周日想做什么特别的活动？"
- "这个周末有什么期待？"

**记录要点**：
```
## 周五家庭连接 [日期]
- 想家时刻：__
- 家庭归属感：__分
- 感恩事项：____、____
- 亲子互动质量：__
- 周末计划想法：__
- 家庭角色认知：__
```

### 周六：自由话题 🎈

**开场**："今天你想聊什么就聊什么！"

**规则**：
- 孩子完全主导话题方向
- 家长只倾听、提问、共情，不评判
- 聊什么都OK：游戏、动画、幻想、同学八卦...

**备用话题**（如果孩子没主意）：
1. "如果你有100万，想买什么/做什么？"
2. "如果变成大人，你想当什么？"
3. "如果变成动物，想当什么动物一天？"
4. "最想去的旅行地是哪里？为什么？"
5. "如果外星人来了，你想送他们什么礼物代表地球？"

**这天的关键是**：让孩子体验"我的话题我做主"

**记录要点**：
```
## 周六自由话题 [日期]
- 孩子主导话题：__
- 孩子兴趣热点：__
- 隐藏需求/担忧：__
- 想象力展现：__
- 家长收获（听到了什么）：
- 下周可跟进的点：
```

### 周日：回顾与展望 🔄

**开场**："我们来给这一周画个句号，再画个问号！"

**复盘问题**：
1. "这一周最开心的事是哪天/哪件事？"
2. "这周学到了什么新东西让你骄傲？"
3. "这周最有挑战的事是什么？现在回头看，容易了吗？"
4. "这周有没有什么遗憾/下次想做得更好？"
5. "如果用一句话总结这一周，你会说？"

**展望问题**：
1. "下周有什么特别的事要发生吗？"
2. "下周最想完成的一个小目标是什么？（具体的事）"
3. "下周想尝试什么新的东西/活动？"
4. "下周想在哪方面进步一点点？"
5. "如果下周有魔法，你最想改变什么？"

**周目标设定**（引导孩子设定1-2个小目标）：
- 学业类："下周我要完成____"
- 习惯类："下周我要每天____"
- 社交类："下周我要和____一起____"
- 兴趣类："下周我要学/做____"

**记录要点**（生成周报时使用）：
```
## 周日回顾展望 [日期]
- 本周高光时刻：__
- 本周挑战：__
- 本周学到：__
- 本周感恩：__
- 下周目标（孩子自定）：
  1. 
  2. 
- 家长支持计划：
- 上周目标完成情况（对照）：
```

## 📊 每周生成物

### 周日晚间（周日对话后）
家长/子代理生成《第X周成长周报》，包含：

1. **本周摘要**：情绪、学业、社交、健康快照
2. **聊天洞察**：从7天对话中提取的关键发现
3. **成长曲线**：情绪稳定性、主动性、创造力等指标
4. **目标达成**：上周目标完成情况（✅/🔄/❌）
5. **下周计划**：基于对话的3个重点行动项
6. **家长反思**： parenting insights，需要调整的沟通方式

周报自动输出到：
`/root/.openclaw/workspace/child-companion/reports/weekly_周报_[日期].md`

## 🔄 child-companion同步流程

### 每日聊天后（立即）
1. 简要记录要点到笔记（或直接回忆）
2. 发送消息给child-companion子代理：
   ```
   @child-companion 记录 [日期] 晚间对话
   主题：周一情绪工作坊
   要点：
   - 心情评分：8/10
   - 开心事：同桌分享贴纸
   - 情绪词汇：兴奋、好奇
   - 观察：能说出2种情绪，识别能力良好
   家长回应：共情并肯定
   ```

### 每周日晚上
1. 查看child-companion汇总的本周聊天数据
2. 生成正式周报（或让子代理自动生成）
3. 保存到钉钉文档（openclaw文件夹）
4. 分享给家人/保存到家庭云盘

## 💡 11个黄金聊天法则

1. **多问开放题** - 避免"是/否"问题，用"什么/怎么/为什么"
2. **少说多听** - 比例：孩子说70%，家长说30%
3. **不评判** - "你这样想很有趣"而非"这不对"
4. **不纠错** - 语法/用词错误私下纠正，聊天时不打断
5. **不比较** - "你和某某比..."是大忌
6. **不 rush** - 给孩子思考时间，沉默10秒再期待回答
7. **不转移** - 孩子说恐龙，不要强行转到学习
8. **分享自己** - 家长也分享当天的1个情绪/1个趣事
9. **身体语言** - 眼神接触、点头、拥抱
10. **幽默应对** - 孩子说蠢话，用幽默化解而非批评
11. **持续记录** - 记录是为了发现模式，不是为了评判

## 🎯 分年龄段调整建议

### 1-2年级（当前适用）
- 问题要具体、具象
- 多用比喻（天气、颜色、动物）
- 每次聊天不超过3个核心问题
- 加入游戏元素

### 3-4年级
- 可加入抽象概念讨论（公平、友谊、未来）
- 增加"假设"问题（如果穿越到古代...）
- 孩子可主导更多话题

### 5-6年级
- 可讨论更深层话题（友谊、学习意义、社会现象）
- 尊重孩子隐私，不强行追问
- 聊天更像平等的成人对话
