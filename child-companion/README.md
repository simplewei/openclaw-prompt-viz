## 首次使用建议

1. **基础信息录入**（请一次性提供）：
   - 孩子姓名、年龄、年级
   - 身高、体重（最近一次）
   - 当前参加的培训班/兴趣班
   - 主要优势和需要关注的点

2. **日常使用流程**：
   - 每天：简单记录语录/趣事/问题
   - 每周五：提供本周总结，子代理生成周报
   - 每月底：审核月报，调整下月计划
   - 每学期：综合评估，长期规划
