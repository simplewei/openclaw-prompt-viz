## 第二步：今晚（第1晚）的对话

### 🎯 如果是周一：情绪工作坊启动

**开场白（试读）**：
```
"宝贝，我们来玩个新游戏！从今天开始，每天晚上我们聊15分钟，这是我们的专属聊天时间。今晚我们要玩'情绪猜猜看'！"

"今天你的心情像什么天气呀？是晴天/多云/小雨/还是打雷？"
```

**流程**（15分钟）：
1. 开场白（1分钟） - 介绍新游戏
2. 情绪天气提问（2分钟） - 引导孩子描述
3. 开心时刻追问（3分钟） - "为什么这么开心？"
4. 小困难了解（3分钟） - "今天有没有觉得难的事？"
5. 明天期待（2分钟） - "明天有什么好玩的事吗？"
6. 总结与拥抱（2分钟） - "今天聊得很开心，妈妈爱你"
7. **记录**（2分钟） - 聊完后立即简单记录

**关键**：第一个晚上不要完美主义，能把流程走完就是胜利！

## 第五步：渐进式改善

### 第1周：流程走通 ✅
- 目标：连续7天都完成聊天（即使只有5分钟）
- 不要求质量，只要求持续性
- 记录用简化模板，聊完就记

### 第2周：增加深度 🔥
- 在2-3个问题后加"为什么"追问
- 尝试1次创意/故事接龙环节
- 开始关注孩子的情绪词汇丰富度

### 第3周：双向分享 🗣️
- 家长也开始分享自己的每天1件小事
- "爸爸妈妈今天也遇到..."（建模脆弱性）
- 把聊天从"采访孩子"变成"双向对话"

### 第4周：定期复盘 📊
- 周日生成第一份正式周报
- 回顾这4周，孩子哪个话题参与度最高？
- 调整下周的话题重点

## 🎯 常见障碍与应对

| 障碍 | 解决方案 |
|------|----------|
| 孩子不想聊 | 降低要求→"今天不想说话？那我们抱5分钟"；用游戏化（抽卡片） |
| 家长忘记 | 设置手机闹钟"晚间聊天"；与孩子的日常routine绑定（如睡前刷牙后） |
| 没话题 | 用本计划表的话题库；从今天学校发的通知/作业切入 |
| 聊成说教 | 家长自我提醒：这是"倾听时间"不是"上课时间"；设置"说教警报" |
| 孩子敷衍 | 家长分享自己有趣的事，建模好的对话；从孩子最爱的话题切入（恐龙/游戏） |
| 无法坚持 | 周日生成周报看到价值后动力回升；与伴侣轮班制；允许自己偶尔跳过 |
