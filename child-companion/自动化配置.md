# 🕐 child-companion 自动化任务配置

## 任务列表

### 1. 周报自动生成（每周日20:00）
```bash
# 使用 system crontab（如果可用）
0 0 20 * * ?  # 每周日20:00（UTC+8对应12:00 UTC，需调整）
```

**替代方案**：OpenClaw cron jobs（优先）
```bash
openclaw cron create \
  --schedule "0 20 * * 0" \
  --task "child-companion-weekly-report" \
  --description "生成孩子成长周报" \
  --command "/root/.openclaw/workspace/child-companion/scripts/generate_weekly_report.sh"
```

### 2. 每日聊天提醒（每晚19:30，周一至周六）
```bash
30 19 * * 1-6  # 周一至周六 19:30
```
提醒内容："今晚的亲子聊天时间到了！主题：__（根据星期轮转）"

---

已创建的脚本：

### scripts/generate_weekly_report.sh
自动扫描本周聊天记录，生成结构化周报，保存到 reports/ 目录

### scripts/publish_weekly_report.sh
将周报发布到钉钉文档（openclaw文件夹），返回链接

---

## 手动触发命令

### 立即生成第N周周报
```
@child-companion 生成第1周周报
```

### 手动发布已生成的周报
```
bash /root/.openclaw/workspace/child-companion/scripts/publish_weekly_report.sh \
  /root/.openclaw/workspace/child-companion/reports/weekly_周报_2026-03-30.md
```

## 状态检查

### 查看子代理状态
```
sessions_list | grep child-companion
```

### 查看本周记录数
```bash
ls -1 /root/.openclaw/workspace/child-companion/memories/*.md | xargs grep -h "【.*聊天" | wc -l
```

### 查看最近一次周报
```
ls -t /root/.openclaw/workspace/child-companion/reports/weekly_周报_*.md | head -1
```
