## 🗺️ 地图标注步骤（高德地图为例）

### 第一步：登录高德地图开放平台
1. 访问：https://lbs.amap.com/
2. 注册/登录（已有账号）
3. 进入"我的数据" → "标点管理"

### 第二步：创建图层
- 图层名：橙发运动周边羽毛球竞品
- 创建成功

### 第三步：逐个添加标记
对每个球馆：
1. 点击"添加标点"
2. 搜索球馆名称或输入地址
3. 选择图标（建议：橙发=红色，竞品=蓝色）
4. 填写属性窗口：
   ```
   名称：球馆名称
   地址：详细地址
   电话：联系电话
   距离：X.X km
   球片数：X片
   平日价格：XX元/小时
   周末价格：XX元/小时
   活动价：XX元
   特色：[选填]
   ```
5. 保存

### 第四步：生成分享链接
- 完成所有标记后
- 点击"分享" → 生成链接/嵌入代码
- 复制链接，可发送给他人查看
