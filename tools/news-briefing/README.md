# 📰 通用新闻简报生成器 (News Briefing Generator)

用户选择领域 + 媒体源 + 推送时间 → 自动生成结构化新闻简报。

## 支持的领域

| 领域 | 说明 |
|------|------|
| 🌍 地缘政治 | 国际关系、外交、冲突、制裁 |
| 💻 科技 | AI、芯片、大模型、科技公司 |
| 📈 经济 | 股市、汇率、通胀、贸易 |
| ⚡ 能源 | 石油、新能源、能源转型 |
| 🏥 健康医疗 | 药物、疫苗、生物科技 |
| 💰 金融 | 央行、数字货币、加密货币 |
| 🌿 气候环境 | 气候变化、碳排放、极端天气 |
| 🛡️ 国防军事 | 军事动态、武器、军演 |

## 支持的媒体

**中文权威**: 新华社、人民日报、央视新闻、财新网、第一财经、澎湃新闻、观察者网、环球时报

**港澳台**: 南华早报、端传媒、联合早报

**英文权威**: Reuters、AP News、Bloomberg、BBC、Financial Times、NYT、WSJ、CNN

**国际**: Al Jazeera、Nikkei Asia、DW、France24

## 推送时间

- 🌅 早间 (08:00)
- ☀️ 午间 (12:00)
- 🌆 晚间 (18:00)
- 🚨 突发追踪 (每4小时)
- 📅 周报 (每周一 09:00)

## 使用方式

### 1. 直接对话生成（推荐）

对 AI 说：
```
帮我生成一份关于[领域]的新闻简报
```

例如：
- "帮我生成一份地缘政治新闻简报"
- "我要科技领域简报，用中文和英文源"
- "生成一份能源周报"

### 2. 配置定时推送

告诉 AI：
```
设置每天早8点推送科技简报
```

### 3. 自定义组合

```
我要一份简报，领域：经济 + 能源
媒体：新华社、Reuters、Bloomberg
时间：每天晚上6点
语言：中英双语
```

## 简报格式

每期简报包含：
1. **📋 今日要闻** — 3-5条最重要新闻，每条含来源、标题、一句话摘要
2. **🔍 深度分析** — 1-2个焦点话题的多源对比分析
3. **📊 数据看点** — 关键数据指标（汇率、油价、指数等）
4. **📌 后续关注** — 未来24-48h值得追踪的事件
