# 使用指南

## 快速开始

只需告诉 AI 你的需求，例如：

### 基础用法
```
"帮我生成一份科技新闻简报"
"我要地缘政治简报"
"来一份能源领域周报"
```

### 自定义媒体源
```
"科技简报，只要 Reuters 和 Bloomberg 的源"
"经济简报，用中文媒体"
"简报用中英双语源"
```

### 定时推送
```
"每天早上8点推送一份科技简报"
"每4小时给我一份地缘政治快讯"
"每周一上午发周报"
```

### 多领域组合
```
"生成一份经济+能源的综合简报"
"科技和国防军事的交叉领域简报"
```

## 领域列表

| ID | 名称 | 关键词示例 |
|----|------|-----------|
| geopolitics | 🌍 地缘政治 | 国际关系、外交、冲突、制裁 |
| tech | 💻 科技 | AI、芯片、大模型、科技公司 |
| economy | 📈 经济 | 股市、汇率、通胀、贸易 |
| energy | ⚡ 能源 | 石油、新能源、光伏、核能 |
| health | 🏥 健康医疗 | 药物、疫苗、生物科技 |
| finance | 💰 金融 | 央行、数字货币、加密货币 |
| climate | 🌿 气候环境 | 气候变化、碳排放、极端天气 |
| defense | 🛡️ 国防军事 | 军事动态、武器、军演 |

## 简报结构

每份简报固定包含4个板块：

1. **📋 今日要闻** — 3-5条核心新闻（来源 + 标题 + 一句话摘要）
2. **🔍 深度分析** — 1-2个焦点话题的多源对比分析
3. **📊 数据看点** — 关键数据指标表格
4. **📌 后续关注** — 未来24-48h值得追踪的事件

## 定时推送设置

AI 会自动创建 cron 任务。支持的时间节点：
- 🌅 早间 (08:00) — 开始一天前了解全局
- ☀️ 午间 (12:00) — 午休时快速浏览
- 🌆 晚间 (18:00) — 收盘/下班后的每日总结
- 🚨 突发追踪 (每4h) — 持续关注热点
- 📅 周报 (周一 09:00) — 每周回顾与展望

## 技术说明

- 基于 web_search + web_fetch 抓取最新新闻
- 支持中英双语媒体源
- 简报自动保存到 `output/` 目录
- 可通过 `config.json` 添加自定义媒体源和领域
